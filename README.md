- Gulp-cli
- Mocha

## Run Locally

### Clone
1. Clone repo
```sh
git clone https://github.com/sallyyeng/fibonacci-generator
```
2. Navigate into repo
```sh
cd fibonacci-generator
```

### Install Dependencies
```sh
npm install
```

### Run Tasks
```sh
npm run build
```

### Start App
```sh
npm run start
```

### Test
```sh
npm run test
```
